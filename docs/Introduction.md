議論の内容について、以下の要件を満たす形で HTML 形式に整形してください。

・私やチームメンバーの「誰が・何を発言したか」が分かること  
・議論の流れを追えること  
・素人の私にも直感的に理解できる表現であること  

表示形式は「吹き出し形式」とし、全体は 3 ペイン構造で、視認性と分かりやすさを重視してください。

・左〜中央ペイン：議論の本編（発言の流れが分かる吹き出し表示）
・右ペイン：用語集
・目に優しくダークモード調にしてください

右ペインの用語集は、アコーディオン形式を使用し、見た目も含めてスタイリッシュに仕上げてください。

また、チームメンバーとして「議事録係」を新たに追加してください。
議事録係は、HTML の最終出力を担当する役割とします。

必要であれば、この議事録係に適したスキル定義を作成しても構いません。
